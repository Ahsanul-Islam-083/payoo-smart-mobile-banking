## 📖 Overview

**PAYOO** is a smart Mobile Financial Services (MFS) interface that simulates a real-world digital banking experience. Built entirely with HTML, CSS, DaisyUI, and Vanilla JavaScript — no frameworks needed — it lets users log in, manage their balance, transfer money, cash out, and track every transaction, all behind a PIN-secured flow.

Think of it as your own lightweight bKash or Nagad, running right in the browser.

## 🚀 Features

- 🔐 **Secure Login** — simple authentication with username and password
- 🏠 **Home Dashboard** — instant overview of account balance and quick-access service buttons
- ➕ **Add Money** — deposit funds into your PAYOO account with ease
- 💵 **Cash Out** — withdraw money securely with PIN verification
- 📤 **Money Transfer** — send money to other users instantly
- 🎁 **Bonus Rewards** — earn and track promotional bonuses on your account
- 📋 **Transaction History** — a complete, scrollable record of every financial activity
- 🔒 **PIN Security System** — every transaction requires PIN authentication for peace of mind
- 📱 **Responsive Design** — works smoothly on mobile, tablet, and desktop via DaisyUI

## 🏃 Run Locally

1. **Clone the repository**
   ```bash
   git clone https://github.com/your-username/payoo.git
   cd payoo
   ```

2. **Open in your browser**
   ```bash
   # Simply open index.html in any browser
   open index.html
   ```
   > No server or build step needed — it's pure HTML, CSS, and JS!

## 🎮 How to Use

| Action | Result |
|---|---|
| Enter username & password | Logs you into the dashboard |
| Click **Add Money** | Deposit funds to your balance |
| Click **Cash Out** | Enter PIN to withdraw money |
| Click **Send Money** | Transfer funds to another user |
| Click **Bonus** | View and claim available rewards |
| Click **Transactions** | Browse full transaction history |
